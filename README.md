# HTML & CSS Assignment Projects

This repository now contains completed mini-projects:

1. **First.html** - HTML fundamentals (headings, lists, table, links).
2. **Formexplain.html** - Bootstrap-based responsive registration form.
3. **Test.html** - Responsive layout with navigation, grid cards, and image stack demo.
4. **nab.html** - Product landing page section with category cards.
5. **Resume/index.html** - Personal resume/portfolio page.

## How to run
Open any HTML file directly in your browser, e.g.:

- `First.html`
- `Formexplain.html`
- `Test.html`
- `nab.html`
- `Resume/index.html`
